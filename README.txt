I. 名称
　PassportUtlからデータを取得するサンプル

II. 説明
　PassportUtlに関する次の処理を行う方法を示すサンプルプログラムです
　・セマフォファイルを介して各種データを取り込む方法

　1. プログラム開発について
　　1) 復号について
　　　本プログラムでは扱っておりません
　　　https://github.com/TechnoVeinsG/PassportUtlSample-Decrypt を参考に、復号してください
　　　このサンプルプログラムでは暗号文ファイルを復号して FileStream を取得していますので、MemoryStream に変換して利用することで本サンプルを容易に暗号文対応に改修できます

　　2) 公開鍵ファイルについて
　　　PassportUtlで使用しているライブラリは Visual Studio に標準添付のものを利用しております
　　　そのため、公開鍵ファイルはXML形式になります
　　　一般的なOpenSSL形式などではありませんのでご注意ください

　　3) その他
　　　本プログラムは処理の概要を簡潔に示すことを目的としているため、エラー処理など割愛している部分が多くあります
　　　本プログラムを参考にして開発される際は、ご注意ください

　2. セマフォファイルおよびデータファイルの削除について
　　セマフォファイルは、パスポートをスキャンするたびに１レコードづつ追加されます
　　しかしセマフォファイルのレコードを削除するタイミングは連携するシステムに依存するためPassportUtlでは行っておりません
　　セマフォファイルおよびデータファイルを削除する場合は、これらのファイルににアクセスしていないタイミングを見はかっらって行ってください

　3. 使い方
　　1) PassportUtlの設定で次の項目を有効にする
　　　・[OCRデータ]-[データ共有]-[データを保存する]にチェックを入れる
　　　・[OCRデータ]-[データ共有]-[ベースフォルダ]に「MRZ」を指定
　　　・[OCRデータ]-[データ共有]-[データファイル]のサブフォルダに「YYYY-MM-DD」を指定
　　　・[OCRデータ]-[データ共有]-[データファイル]のファイル名に「HH-mm-SS-Identifying」を指定
　　　・[可視光画像]-[データ共有]-[データを保存する]にチェックを入れる
　　　・[可視光画像]-[データ共有]-[ベースフォルダ]に「Color」を指定
　　　・[可視光画像]-[データ共有]-[データファイル]のサブフォルダに「YYYY-MM-DD」を指定
　　　・[可視光画像]-[データ共有]-[データファイル]のファイル名に「HH-mm-SS-Identifying」を指定
　　2) セマフォファイルを指定する
　　3) ADR300 でパスポートをスキャンする
　　PassportUtlの設定に従ってOCRデータおよび可視光画像が表示されます
